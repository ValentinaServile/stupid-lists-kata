# Stupid Lists Kata

### Goal:

Sort a list of numbers so that the < and > signs are alternating between elements.

### Example:

[1,2,3,4,5,6] => [1,4,2,5,3,6]

because

1 < 4 > 2 < 5 > 3 < 6

## Requirements

- python
